¿Qué es Bootstrap?

Bootstrap es un framework gratuito de código abierto para la creación de páginas y aplicaciones web. Diseñado para permitir el desarrollo receptivo de páginas web orientada a teléfonos, Bootstrap proporciona una colección de sintaxis para modelos de diseño.

Como un Framework, Bootstrap incluye los conceptos básicos para el desarrollo web receptivo, por lo que los desarrolladores solo necesitan insertar el código en un sistema de cuadrícula predefinido. El framework Bootstrap se basa en lenguaje de marcado de hipertexto (HTML), hojas de estilo en cascada (CSS) y JavaScript. Los desarrolladores web que usan Bootstrap pueden crear sitios web mucho más rápido sin perder tiempo preocupándose por los comandos básicos y funciones.

¿Para que sirve Bootstrap?

Bootstrap convierte el diseño web receptivo en realidad. Hace posible que una página web o aplicación detecte el tamaño de la pantalla del visitante y orientación y automáticamente adapta la pantalla en consecuencia. El enfoque móvil asume teléfonos inteligentes, tabletas y aplicaciones móviles para tareas específicas son las principales herramientas de los empleados para realizar el trabajo. Bootstrap aborda los requisitos de aquellas tecnologías e incluye componentes de interfaz de usuario, diseños, herramientas de JavaScript y la implementación del framework. El software está disponible precompilado o como código fuente.
